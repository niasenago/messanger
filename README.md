### Vilnius University Java OOP Course Assignment ###  
This assignment, part of the Object-Oriented Programming course at Vilnius University, involves creating client-server communication for enabling message exchange among different clients with the following features:

- Ability to establish client-server communication.
- Option to create rooms for group communication.
- Ability to exchange messages with all users in a room.
- Capability to exchange messages with each user individually.
- Messages, rooms, and users should be stored in a file (file type is freely selectable, e.g., JSON, CSV, TXT, etc.).
- Server-side implementation can utilize ServerSocket, while the client-side can use Socket.
- Graphical User Interface (GUI) can be implemented using Swing or JavaFX.
